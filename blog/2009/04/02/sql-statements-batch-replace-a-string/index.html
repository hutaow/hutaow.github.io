SQL语句实现某一字符串的替换

Date 四 02 四月 2009 By hutaow Category Tech Tags Database / SQL / MySQL / 数据库

使用update+replace语句：

mysql update table_name set column_name = replace(column_name,’hutaow_original_string’,'hutaow_new_string’);

其中，table_name是表的名字，column_name是要替换数据的列的名字，hutaow_original_string是待替换的原始字符串，hutaow_new_string是新字符串，MySQL下验证成功。
